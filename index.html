Sobre Angular

Es un framework para desarrollar aplicaciones en Javascript de un modo organizado y con mejor fluidez de interacción.

Sobre Typescript

Un lenguaje superset que brindan cualidades como tipado, interfaces y clases a Javascript encontradas en C# y Java.

Audiencia

Cualquier profesional de sistemas que desarrolla aplicaciones web empresariales o para consumidores y/o web designers o arquitectos de software que necesitan manejar o interactuar con equipos de desarrollo.

Requisitos

Tener conocimientos de HTML, CSS, Javascript y al menos un framework de server como ASP.NET, PHP o Ruby on Rails.
